# Topic Modeling Plastik
Penelitian ini menganalisis 142 artikel berita relevan yang difilter dari 1.541 artikel sumber media nasional seperti Kompas.com, CNN Indonesia, Bisnis.com, dan VIVA.co.id. Analisis dilakukan untuk memetakan bagaimana media mengonstruksi realitas krisis industri yang dipicu oleh guncangan pasar global
